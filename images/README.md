# Photos du site

Dépose tes JPG ici avec ces noms exacts. Tant qu'un fichier manque,
le site retombe sur une image Unsplash similaire (via `onerror`) — donc
tu peux les ajouter une par une sans rien casser.

| Nom du fichier      | Sur le site                              | Conseil               |
| ------------------- | ---------------------------------------- | --------------------- |
| `terrasse.jpg`      | Hero plein écran + galerie               | Format paysage, large |
| `cuisine.jpg`       | Galerie (la grande)                      | Paysage               |
| `chambre.jpg`       | Section « Chez nous » + galerie          | Portrait OK           |
| `coworking.jpg`     | Galerie                                  | Paysage               |
| `bois.jpg`          | Galerie (forêt / vue fenêtre)            | Paysage               |

Taille recommandée : 2000 px sur le plus grand côté, JPG qualité 80,
< 400 Ko par image (passe-les dans https://squoosh.app si besoin).
